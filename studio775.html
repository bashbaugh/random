Welcome To Studio 775


Welcome to Studio 775, an new mini makerspace and a collaborative, community-driven space where users of all ages can learn to use new tech tools in creative ways. Below are some links to popular and common websites and resources to use in here :)


Tinkercad

A cloud based, easy drag and drop website for designing 3d models. If you have never used CAD software before, this might be a good place to start - after creating an account, you will be redirect to a short tutorial that will teach you the bascis.

Tinkercad.com

Thingiverse
